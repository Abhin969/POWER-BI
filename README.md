# Power BI Data Analytics 

This repository contains multiple interactive dashboards created using Power BI.
The projects focus on real-world business scenarios such as HR analytics, sales performance, and audiobook analysis.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV datasets
- Data Modeling & Visualization

## Projects Included

### 1. Recruitment, Performance & HR Insights Dashboard
- Analyzed employee data including salary, performance score, attendance, and attrition
- Key insights:
  - Employee distribution by department and gender
  - Performance score comparison across departments
  - Absence trends by citizenship
  - Recruitment source effectiveness

### 2. Audiobook Performance Dashboard
- Analyzed audiobook data based on ratings, prices, authors, narrators, and year
- Key insights:
  - Top authors and narrators by book count
  - Rating and pricing trends
  - Growth of audiobooks over time

### 3. Sales Performance Dashboard
- Analyzed sales, profit, and regional performance
- Key insights:
  - Profit trends over time
  - Top-performing products
  - Regional sales distribution

## Key Skills Demonstrated

- Data modeling
- KPI creation using DAX
- Business-focused storytelling with dashboards
- Interactive slicers and filters
